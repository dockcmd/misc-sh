# Misc Docker Command Shell

Misc commands that wrap existing docker containers.


# Shln

Any commands must be installed manually:

```bash
shln install github.com/dockcmd/misc-sh v0.0.2

# install manually
shln ln gcloud
shln ln npm
...

```
